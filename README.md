# Backend Stack Demo

This project provides samples of using of the main modern Java technology stack required for backend engineers. 

The idea is to keep it simple as much as possible.

So far, we have these as listed:

* Kafka
* RabbitMQ
* Spring WebFlux
* Gitlab Pipeline
* 100% Test Coverage
